# Homepage Wireframes (Rebrand)

## Goals

- Make the value proposition understandable in < 5 seconds
- Create a single primary journey: **Signup → Developer portal → First API call**
- Provide secondary journeys: **Pricing → Purchase** and **Login**
- Establish trust with security proof points and product breadth
